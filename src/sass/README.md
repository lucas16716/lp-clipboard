# AXIS — Sass Architecture

This directory contains the Sass architecture used by AXIS.

**Structure:**

- `abstracts/`
  Functions, mixins and design tokens.

- `base/`
  Global styles, resets and typography.

- `layout/`
  Layout utilities such as container, flex and grid.

- `components/`
  Reusable neutral interface components.

- `sections/`
  Project-level sections.
